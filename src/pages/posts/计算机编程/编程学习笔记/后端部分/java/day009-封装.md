# **🎯 【标准封装类模板】（最新版！一眼对齐 day009 要求）**

```
package cn.itsource.assignment;

/**
 * 学生类，演示标准封装
 */
public class Student {
    // 1. 成员变量私有化
    private String name;
    private int age;
    private String gender;

    // 2. 无参构造方法
    public Student() {
        this("无名氏", 0, "未知"); // 使用 this(...) 调用有参构造器，统一初始化逻辑
    }

    // 3. 有参构造方法
    public Student(String name, int age, String gender) {
        this.name = name;
        this.age = age;
        this.gender = gender;
    }

    // 4. Getter 和 Setter 方法
    public String getName() {
        return this.name;
    }

    public void setName(String name) {
        this.name = name;
    }

    public int getAge() {
        return this.age;
    }

    public void setAge(int age) {
        this.age = age;
    }

    public String getGender() {
        return this.gender;
    }

    public void setGender(String gender) {
        this.gender = gender;
    }
}
```

# **📋 【标准测试类模板】（对应 StudentTest.java）**

```
package cn.itsource.assignment;

public class StudentTest {
    public static void main(String[] args) {
        // 使用无参构造 + setter 赋值
        Student student1 = new Student();
        student1.setName("李华");
        student1.setAge(20);
        student1.setGender("男");
        
        System.out.println(student1.getName());
        System.out.println(student1.getAge());
        System.out.println(student1.getGender());

        // 使用有参构造直接赋值
        Student student2 = new Student("王芳", 21, "女");

        System.out.println(student2.getName());
        System.out.println(student2.getAge());
        System.out.println(student2.getGender());
    }
}
```

# **⚡【总结：你先前犯过的 5 个典型错误】**

|**错误**|**错误例子**|**正确做法**|
|---|---|---|
|1. 构造方法错误加了 void|public void Student(...) ❌|public Student(...) ✅|
|2. getXxx() 写成了 getXxx(参数)|getGender("男") ❌|应该是 getGender() ✅|
|3. 中文全角引号问题|“男” ❌|使用英文半角引号 "男" ✅|
|4. 成员变量暴露，未封装|public String name; ❌|private String name; ✅|
|5. 测试类调用 set 方法当成 get 方法|System.out.println(setGender()); ❌|正确用 getGender() ✅|
