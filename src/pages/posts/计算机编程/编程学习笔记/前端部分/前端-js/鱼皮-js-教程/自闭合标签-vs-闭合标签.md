### 闭合标签（有内容）

```jsx
<Link to="/todos">Todo 列表</Link>
//  开始标签 ↑        ↑ 内容   ↑ 闭合标签（/ 在标签名前）
```

有**开始标签**和**结束标签**，中间可以包裹文字或子元素。

---

### 自闭合标签（无内容）

```jsx
<Link to="/todos" />
//                 ↑ 斜杠在 > 前，代表自己闭合自己
```

没有子内容时使用，`/` 写在 `>` **前面**。

---

### 本质区别

| | 写法 | 能包裹内容？ |
|--|------|------------|
| 闭合标签 | `<Tag>内容</Tag>` | 能 |
| 自闭合标签 | `<Tag />` | 不能 |

---

### 你代码里的问题

```jsx
<Link to="/todos">Todo 列表<Link/>
//                          ↑ 这是又开了一个新的 <Link> 自闭合标签
//                            而不是在关闭前面那个 <Link>
```

`<Link/>` 不是闭合标签，它是**另一个新的自闭合 `<Link>` 标签**，所以第一个 `<Link to="/todos">` 永远没有被关闭，导致报错。

正确写法：闭合标签的斜杠必须在**标签名前面**：

```jsx
<Link to="/todos">Todo 列表</Link>
//                          ↑ </Link> 才是闭合标签
```
