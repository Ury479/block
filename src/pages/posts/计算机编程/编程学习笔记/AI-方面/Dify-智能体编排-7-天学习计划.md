## ⚠️ 常见坑与对策

| 坑 | 表现 | 对策 |
|---|------|------|
| Dify 自部署 | Docker 装不上 | 先用 SaaS 版（dify.ai），熟悉后再自部署 |
| 知识库没效果 | 召回率低 | 调 Chunk Size + 用 Re-rank 模型 |
| Agent 循环 | 永远停不下来 | 设置 max_iteration + 加终止条件 |
| Token 超费 | 账单爆炸 | 选小模型 + 限制 max_tokens + 监控用量 |
| 工作流报错 | 节点连不上 | 善用"调试模式"+ 单步执行 |
| 多语言支持差 | 中文文档不全 | 加入 Dify Discord/GitHub Issues |
