`<div class="text-center font-sans text-gray-700 antialiased">`
- **`text-center`**：页面内容文字居中；
    
- **`font-sans`**：应用了无衬线字体（默认 Tailwind 的 UI 字体）；
    
- **`text-gray-700`**：文字呈现为较深的灰色（非纯黑，柔和）；
    
- **`antialiased`**：字体进行了抗锯齿优化，显示更清晰；
